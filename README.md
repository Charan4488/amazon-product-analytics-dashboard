# Amazon Product Analytics Dashboard

## Project Overview

This project is an interactive Power BI dashboard that analyzes Amazon product data to provide insights into pricing, discounts, ratings, reviews, and category-wise performance.

## Features

- KPI Cards
  - Total Products
  - Total Reviews
  - Average Rating
  - Average Discount
  - Average Discounted Price

- Category-wise Product Distribution

- Average Rating by Category

- Average Discount by Category

- Customer Reviews Analysis

- Product Explorer

- Price vs Rating Analysis

## Tools Used

- Power BI
- Microsoft Excel
- DAX

## Dataset

Amazon Product Dataset

## Dashboard Pages

1. Overview Dashboard
2. Category Analysis
3. Customer Reviews
4. Price vs Rating Analysis
5. Product Explorer
